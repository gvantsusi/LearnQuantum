# Project Description
This project is a Java-based desktop application developed using the Swing framework. The application supports topic-based learning and assessment through quizzes, while maintaining persistent records of user progress using local storage.
# Functionality
The system provides a learning mode for content review and a quiz mode for self-assessment. User progress and quiz results are stored locally, enabling continuity between sessions.
# Technologies Used
Java,
Java Swing,
Object-oriented programming principles
# Execution
The application can be executed by running MainApp.java in a standard Java development environment.
# System Structure
The application follows a modular design, separating user interface components, quiz logic, data models, and persistence mechanisms. Key classes include QuizManager, ProgressStorage, and UserProgress.
